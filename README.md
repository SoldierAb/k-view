# `|` A high quality UI Toolkit built on Vue.js


## Install
```shell
npm i @cgj/k-view -S
```

## Quick Start
``` javascript

import KView from '@cgj/k-view'
import '@cgj/k-view/lib/kview/style.css'
Vue.use(KView);

// or
import {
  Loading,
  Tree
  // ...
} from 'element-ui'

Vue.use(Loading);
Vue.use(Tree);
```
